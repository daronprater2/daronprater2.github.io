HomeWhere

Welcome to HomeWhere - The personalized recommendation system for living in Chicago.

HomeWhere has two main features: View By Metric and View By Total. Both of these features are described in more detail below.

  • View By Metric - When in the view by metric mode you have the ability to compare Chicago neighborhoods across a variety of metrics. To use the View By Metric mode, follow these steps:

    1. Select "View by Metric" from the Application Mode dropdown.
    2. Select the Metric of interest from the metric menu.
    3. Colors on the map indicate whether a zip code is "Good" or "Bad" for that metric. The meaning of the color scales for each metric are defined to the right under the "Color Scales for each Metric" heading.
    4. Move the "Years in the Future" slider in the Time menu to see how the zip codes look into the future.

  • View By Total - When in the view by total mode HomeWhere will recommend a neighborhood that best suits you based on your preferences. A neighborhood shaded green means HomeWhere predicts you would like living in that area. A neighborhood shaded red means HomeWhere predicts you would not like living in that area. When using the View By Total mode follow these steps:

    1. Select "View by Total" from the Application Mode dropdown.
    2. Answer the questions.
    3. HomeWhere then finds the zip codes you would like living in the most based on your answers.
    4. Move the sliders in each category to adjust your preferences.
    5. The farther right the slider indicates the more you care about the metric. For example, sliding Median Home Price all the way to the left indicates you prefer higher home prices.
    6. Click refresh map.
    7. Move the slider in the Time menu to see how your preferred zip code would change in the future.

Additional Features and Notes

  • Zip Code Statistics - Hover over a zip code to see the statistics for that zip code.
  • The Education scores measure the overall quality of schools in that zip code and are measured on a scale from 1-5.
  • Zip code 12311 is a combination of zip codes 60601, 60602, 60603, 60604, 60605, and 60611.
  • Zip code 6761 is a combination of zip codes 60606, 60607, 60661.
  • Moving the years into the future slider makes predictions for each zip for the years ahead specified.
